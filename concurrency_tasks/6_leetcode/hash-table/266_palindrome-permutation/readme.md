Описание задачи:

Проверьте, может ли перестановка символов строки образовать палиндром.

Пример:
Вход: "code"
Выход: false

Вход: "aab"
Выход: true

Вход: "carerac"
Выход: true

Условие:
Палиндрому может соответствовать строка, в которой не более одного символа встречается нечетное количество раз.
