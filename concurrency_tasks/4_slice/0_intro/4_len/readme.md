В этом примере мы пытаемся узнать пустой ли массив. По логике, если  в выражении arr == nil мы получаем true, значит массив пустой.
В первом случае true, а во втором false. 
Почему так происходит?
В первом случае у нас слайс не проинициализирован, а во втором мы переменую arr проинициализировали пустым слайсом.
Из этого следует вывод, что проверять пустой ли слайс через проверку на равенство с nil некорректно. Для этого стоит использовать len(arr) == 0.
